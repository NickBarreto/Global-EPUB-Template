The break class with a single astersik above should be used in every instance of a scene break. In print books, the asterisk is only used when the scene break appears at the bottom of a page. Otherwise it is merely denoted by a blank space between paragraphs. The next paragraph after the space is not indented as this one.

In our ebooks, every time there is a break the asterisk should be included. Even if there wasn't one in the print book. Some books will use a different character or image in the break scenes, and if this is the case, that character or image should replace the asterisk in every instance.

If there is an image in the text, this is the default way to present it. Be careful that images that are significantly smaller than a kindle's screen will appear pixelated unless you explicitly include that image's pixel height and width as attributes in the img element.
